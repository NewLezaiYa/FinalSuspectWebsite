FinalSuspect
输出日志

This feature provides a simple log management solution：

  1. One-Click Export: Automatically copies log files from the current Among Us\BepInEx directory to a dedicated directory specified by the mod.
  2. Quick Access: Automatically opens the target folder window immediately after copying is complete, making it easy to view log content.
  3. Auto-Clean: Original log files are automatically cleared every time Final Suspect starts, preventing excessive space usage.

The log folder is located at the following path ：

%APPDATA%
\..\LocalLow\Innersloth\Among Us\FinalSuspect\Logs\Final Suspect-logs

You can press Win + R to open the Run dialog, paste this address, and confirm to manually access the directory.
