# FinalSuspect Website Project

FinalSuspect is a website project dedicated to providing game mod guides and feature showcases, tailored for the Among Us player community, particularly users interested in mods.This project helps users quickly understand the mod's features, installation methods, and related configurations through an intuitive interface and well-structured content.

[English](README.en.md) [Traditional Chinese](README.tw.md)

## Usage Instructions

- **Homepage** (`index.html`)：Provides an overview of the project and serves as the entry point.
- **Features Page** (`Features/`)：Showcases specific mod functionalities, such as music control and log management.
- **Options Page** (`Options/`)：Provides configuration options for the mod, such as auto-ending games and disabling vanilla sound effects.
- **Guide Page** (`Guide/`)：Provides detailed instructions for mod installation, updates, and log output methods.
- **FAQ Page** (`FAQ.html`)：Frequently Asked Questions to help users quickly resolve common queries.
