FinalSuspect Q&A

Is Final Suspect compatible with other mods?

No consideration, no adaptation, no compatibility.

The latest version does not have an installation package. How should I install it?

You can download the installation package of the previous version and then install Final Suspect as usual.
Final Suspect should then prompt you to update; simply follow the on-screen instructions.

If no prompt appears (possibly because you are on the OpenBeta), you can manually update by following the update tutorial.

Among Us client is showing an error.

如果您将Final Suspect安装在了官方目录，可以直接通过Steam来打开Among Us。

无法连接到Final Suspect服务器

请检查你的网络。

在1.1_20250501 - 后续的2个OpenBeta版本中，无法连接服务器实际上是一个Bug。
该问题会在后续的OpenBeta版本中修复。

无法保证新版兼容性，即使其是Final Suspect的最新版本。
