There is no installation package for the latest version,gow should I install it?

You can download the installation package for the previous version and then install Final Suspect as usual.
After that, Final Suspect should prompt you to update; simply follow the on-screen instructions.

If no prompt appears (likely because you're on the OpenBeta ) , you can manually update by following the tutorial.

The compatibility of older versions of Final Suspect with newer versions of Among Us is generally poor and not recommended.

Compatibility cannot be guaranteed with the new version, even if this is the latest version of Final Suspect.
But we believe that once someone in China invents and popularizes time travel technology, we'll be able to adapt Final Suspect before Innersloth even releases the new version of Among Us.
