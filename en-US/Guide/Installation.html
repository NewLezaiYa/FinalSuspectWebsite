Steam installation

Default installation path reference：

C:\Program Files (x86)\Steam\steamapps\common\Among Us

Quick opening method：Press the Win + R key combination, enter the path and press Enter

Steps：

  1. In the Steam client, right-click 「Among Us」 → 「select Properties」 → switch to the 「Local Files tab」 → click 「Browse」 to open the Game root directory

Epic Install

Default installation path reference：

C:\Program Files\Epic Games\AmongUs

Quick method: In the Epic Games client library, locate 《Among Us》 → click the “⋯” on the right → select “Show in Explorer.”

Steps：

  1. Open the game’s root directory on the Epic platform using the method described above.
  2. Extract the downloaded Final Suspect archive to the game’s root directory. The final file structure should be the same as on the Steam platform.
C:\Program Files\Epic Games\AmongUs\BepInEx\plugins\FinalSuspect.dll

Epic Platform restrictions

Must be installed in the official Epic game directory (using a non-official path will cause verification to fail)
