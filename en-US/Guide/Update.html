FinalSuspect Update

Automatic Update

Final Suspect has a built-in update checker within the game, and will automatically check for updates each time you launch it.
If an update is available, you will be required to update; otherwise, you cannot continue playing.

The update check logic differs between Release and OpenBeta versions：
Release versions will not prompt you to update to an OpenBeta version, while OpenBeta versions do not perform update checks.

Manual Update

If not necessary, we recommend that you choose automatic update.

If the update check fails or issues occur during download, you can manually update by following these steps：

  1. 访问GitHub Releases，下载最新版本的dll文件。
  2. 打开Among Us根目录，再继续打开BepInEx\plugins文件夹。
  3. 将下载的dll文件替换掉原有的dll文件。
  4. 至此，更新结束。
