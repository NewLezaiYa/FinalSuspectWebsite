Auto-Export

You can select the output log in the Final Suspect function, and then Final Suspect will open the folder where the output log is located.

For an introduction to the automatic log export function, please refer to this document.

The output directory of Final Suspect is located in the following path：

%APPDATA%
\..\LocalLow\Innersloth\Among Us\FinalSuspect\Logs\Final Suspect-logs

You can pressWin + R, and paste this path to open it manually.

Manual View

If you can't open the settings page, you can also choose to view the log manually.

启动Steam，在库页面中选择Among Us，点击右侧的设置图标，选择管理，随后选择浏览本地文件。Steam will open to the Among Us root directory.

Open the BepInEx folder, and the LogOutput.log file is the log file.This file will be automatically cleared when you restart FinalSuspect.This file will be automatically cleared when you restart Final Suspect.
