Quick Start Mode

In the Final Suspect mod, we introduced Quick Start Mode to optimize the game loading process.When this mode is enabled, the system will skip non-essential steps.

In testing, a normal startup takes about 15 seconds, while with Quick Start enabled it takes only 5 seconds to reach the main menu.This mode is especially suitable for testing scenarios that require frequent game restarts, and players can switch modes via the settings menu.

A full resource load is still required during the first installation or major version updates.
