# FinalSuspect 網站專案

FinalSuspect 是一個專注於提供遊戲模組指南與功能展示的網站專案，適用於 Among Us 玩家群體，尤其是對模組有興趣的使用者。本專案透過清晰的介面與結構化的內容，幫助使用者快速了解模組的功能、安裝方式以及相關配置。

[English](README.en.md) [简体中文](README.md)

## 使用說明

- **​首頁**​ (index.html)：提供專案的整體介紹與入口。
- **功能页面**​ (Features/)：展示模組的具體功能，如音樂控制、日誌卸載等。
- **选项页面**​ (Options/)：提供模組的配置選項，例如自動結束遊戲、停用原版音效等。
- **教程页面**​ (Guide/)：詳細介紹模組的安裝、更新和日誌輸出方式。
- **疑難解答页面**​ (FAQ.html)：常見問題解答，幫助使用者快速解決疑問。
